# Alter Ego v1.0.1.3

## Changes

- **Deleted characters now disappear from the list.** Refreshing re-syncs the character roster from Hoard & Seek and removes any character that no longer exists, so characters deleted in-game stop showing up. Previously they lingered indefinitely.

## Install

Copy `AlterEgo.dll` into your `<GW2>/addons` folder.
