# Alter Ego v1.0.2.0

## Changes

- **Hoard & Seek v4 compatibility.** Updated to the latest Hoard & Seek cross-addon interface. When H&S is briefly busy, queries now back off and retry automatically instead of silently failing, and the old permission-approval prompts (no longer used by H&S) have been removed.
- **Pie UI integration.** When Pie UI is installed, build codes shared in chat appear as clickable links; clicking one offers it for import into your Build Library. Full builds (gear, runes, sigils, relic, game mode) are carried across.

## Install

Copy `AlterEgo.dll` into your `<GW2>/addons` folder.
