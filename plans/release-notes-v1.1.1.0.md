# AlterEgo v1.1.1.0

## New: Fashion / wardrobe sharing

Right-click any equipment tab for a **Fashion** menu:

- **Copy Code** — copies a native GW2 wardrobe template chat link for that tab's armor and weapons (skins + dyes, using the transmuted skin or the item's default appearance). Paste it in chat to share your look.
- **Preview in Pie UI** — when Pie UI is loaded, opens its native Wardrobe Template window for that look. Hidden when Pie UI isn't running.

Single-tab characters now show their equipment tab chip, which doubles as the right-click target.
