# AlterEgo v1.1.0.0

## Build Creator & Editor
Create builds from scratch in the Build Library and edit them in place on the build card — specializations, traits, skills, gear, Revenant legends, and Ranger pets. Save/Cancel with unsaved-change protection so cancelling reverts cleanly.

## TrueType Font Support
Pick a crisp TrueType font for all AlterEgo text, at any size, from the addon settings. Use the bundled **Inter** face (downloaded on first use) or drop your own `.ttf` into the addon's `fonts/` folder. Defaults to the standard Nexus font — nothing changes unless you opt in.
