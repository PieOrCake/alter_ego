# Alter Ego v1.0.1.1

## Changes

- **Raid bounty banners.** Strike-mission bounties (like "Voice and Claw of the Fallen") now show their proper strike banner and "Strike Mission · <zone>" subtitle, matching the Weekly Strikes section. Raid-wing bounties now prefix the wing number, e.g. "W7 · The Key of Ahdashim".
- **Build library export/import.** Full backup and restore of your saved builds to a JSON file. Import offers Merge (skip duplicates) or Replace All.
- **Recommended fractals.** The REC chip row now shows fractal names instead of raw scale numbers (e.g. "Underground Facility (31)").
- **Bugfixes.** Build library rename/notes no longer silently truncate. Settings loader tolerates malformed fields. Achievement saving no longer briefly stalls the UI.
